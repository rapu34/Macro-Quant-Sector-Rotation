# Experiments

Sandbox for sensitivity studies, validation scripts, and non-production outputs.

## Structure

- `scripts/` — Experiment scripts (crisis false positive, elasticity, expansion, validation)
- `outputs/` — Outputs from experiment scripts
- `data/` — Extended data (e.g. `*_EXTRA.csv`)

## Production outputs (in `/outputs`)

- backtest_report.md
- final_risk_governance_report.csv
- out_of_sample_test.csv
- hmm_regime.csv
- model.pkl
